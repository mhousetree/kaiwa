# kaiwa

卒業研究で制作した外国語会話集アプリケーションです。
卒論や発表スライドをご覧になりたい方はお問合せください。

# frontend

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

# backend

## Project setup
```
pip install -r requirements.txt
```

## Compiles and hot-reloads for development

```
uvicorn main:app --reload
```
